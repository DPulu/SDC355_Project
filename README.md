# My Portfolio

This portfolio page was created to showcase personal projects and practice core web development skills.

## Languages and Technologies Used

- HTML
- CSS
- JavaScript
- DOM manipulation
- sessionStorage
- localStorage
- GitHub Pages

## Features

- Dynamic project rendering using JavaScript objects
- Project data stored and retrieved using sessionStorage
- Dark mode preference saved with localStorage
- Contact form interaction
- Tooltip on the submit button
- Welcome modal
- Featured content section
- Skills list generated with JavaScript

## Dependencies

No external dependencies are required. The page runs in a modern web browser.

## Required Files

- `index.html`
- `styles.css`
- `Project1.png`
- `README.md`
